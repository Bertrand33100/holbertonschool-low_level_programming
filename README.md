# C - Low Level Programming

Ce dépôt regroupe l'ensemble de mes projets réalisés dans le cadre de la formation **Software Engineering** à **Holberton School**.

L'objectif de ces projets est de construire des bases solides en programmation système à travers le langage C, la gestion de la mémoire, les structures de données, les algorithmes et les outils de développement sous Linux.

---

# Objectifs pédagogiques

Au cours de ce dépôt, j'apprends à :

- Comprendre le fonctionnement d'un programme en mémoire
- Écrire du code propre, lisible et modulaire
- Développer en respectant les normes Betty
- Utiliser Git et GitHub dans un environnement professionnel
- Déboguer efficacement avec GDB et Valgrind
- Résoudre des problèmes algorithmiques
- Manipuler la mémoire de manière sûre
